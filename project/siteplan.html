Site Plan

Site Name

Harare Heritage Hub

This name represents a site dedicated to showcasing the cultural, historical, and community landmarks of Harare, especially Eastview and Epworth. It was chosen to highlight local pride and visibility.

Optional domain availability: harareheritage.org

Site Purpose

The site provides a hub for local history and culture by offering information on temples, landmarks, and community projects. It also includes resources for visitors and residents, such as event listings and contact forms.

Scenarios

  • Where can I find information about historic temples in Epworth?
  • How do I contact the community organizers for upcoming cultural events?

Color Schema

Primary Color: #2C3E50 (used for headings and navigation)

Accent Color: #E67E22 (used for buttons and highlights)

Background Color: #F4F4F4 (used for page background)

Typography

Heading Font: "Montserrat", sans-serif

Body Font: "Roboto", sans-serif

Headings will use Montserrat for a bold, modern look. Body text will use Roboto for readability.
